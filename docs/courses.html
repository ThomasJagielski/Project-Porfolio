COURSES

Quantitative Engineering Analysis

Quantitative Engineering Analysis is a two semester course that introduces topics in math and physics. Topics introduced include linear algebra, multivariable calculus, and differential equations. Tools used during this course included MATLAB and Mathematica.

Software Design

This course introduced basic coding principles through Python.

Introduction to Sensors, Instrumentation, and Measurement

This course introduced concepts in electrical sensing with an emphasis on analog circuitry.


Principles of Engineering

Principles of Engineering introduces electrical, mechanical, and software integration through a semester long project with a team of five people.

Discrete Math

This course introduced principles in discrete math such as counting techniques, types of functions, proof writing, combinatorics, induction, recurrance relations, and graph theory.

User Oriented Collaborative Design

User Oriented Collaborative Design introduces design techniques for user inclusive design. This course included a semester long project, and thus, also emphasized topics in project management.


Introduction to Microelectronic Circuits with Laboratory

This course introduced topics in linear circuits, BJT and CMOS transistor design, and the basic building blocks of analog integrated circuits. We learned about single stage transistor amplifiers, current mirrors, cascodes, differential pairs, and single-stage operational amplifiers. This course focused on design-oriented circuit analysis and reasoning skills.
